# Sakina (سكينة) - Minimalist Islamic UI Challenge 

##  Description
**Sakina** is a minimalist, spiritually calming web interface designed to showcase beautiful Quranic verses and Islamic remembrances (Athkar). The main focus of this challenge was to master layout alignment using advanced **CSS Flexbox** and to implement smooth, modern **Box-Shadow** properties for a clean card-based design.

The layout is crafted with a soft, muted pastel color palette to ensure a peaceful user experience and maximum visual comfort.

##  Live Demo
Explore the live website here: (https://nadaalafify.github.io/Sakina-Islamic-UI-Flexbox/)

##  Tech Stack
- **HTML5:** Semantic components to build structured layout blocks for cards, text wrappers, and action buttons.
- **CSS3 (Pure CSS):** Leveraging **Flexbox** for precise axis alignment, custom grid responsiveness, smooth hover transitions, and refined box-shadow aesthetics.

##  Key Features & Design Layout
- **Aesthetic Card Design:** Beautifully aligned cards featuring Quranic verses, complete with metadata (Surah name, Ayah number, and total Ayah count).
- **Interactive Remembrances:** Symmetrically aligned bottom buttons for Athkar triggers.
- **Flexbox Mastery:** Developed completely from scratch without external grid systems to ensure pixel-perfect vertical and horizontal centering.
- **Icon Navigation Bar:** A clean footer section holding navigation icons centered smoothly via flex containers.
